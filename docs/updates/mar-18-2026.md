## Specific Feature Highlights

### 1. Deals & Bundles - Category-Specific Variations
**Problem**: Previously, deals allowed manual input for sizes and units regardless of category.

**Solution**: 
- Variations now automatically adapt to category attributes
- "Size per unit" uses category's denominations (e.g., 0.125, 0.25, 0.5, 1)
- "Unit" field is now a select dropdown populated from category's units (e.g., oz, g, ml)
- When category changes, variations update to match new category's options
- Maintains user input for totals and custom labels

**Benefits**:
- Reduced errors from manual entry
- Consistent pricing across categories
- Better UX with pre-populated options
- Data consistency between deals and products
